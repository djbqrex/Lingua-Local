## Code Flow: Streaming Response

### Backend (Python)

```python
# backend/app/routes/conversation.py

async def speak_and_respond_stream(...):
    async def event_generator():
        # 1. Transcribe audio
        transcription = stt_handler.transcribe(audio_path)
        yield f"event: transcription\ndata: {json.dumps(transcription)}\n\n"
        
        # 2. Stream LLM response
        full_response = ""
        for chunk in llm_handler.generate_stream(messages):
            full_response += chunk
            yield f"event: response_chunk\ndata: {json.dumps({'chunk': chunk})}\n\n"
        
        # 3. Complete
        yield f"event: complete\ndata: {json.dumps({'full_response': full_response})}\n\n"
    
    return StreamingResponse(event_generator(), media_type="text/event-stream")
```

### Frontend (JavaScript)

```javascript
// frontend/js/api.js

static async speakAndRespondStream(audioBlob, ..., callbacks) {
    const response = await fetch('/api/conversation/speak-stream', {
        method: 'POST',
        body: formData
    });
    
    const reader = response.body.getReader();
    const decoder = new TextDecoder();
    
    while (true) {
        const { done, value } = await reader.read();
        if (done) break;
        
        const data = JSON.parse(decoder.decode(value));
        
        if (data.text) {
            callbacks.onTranscription(data);
        } else if (data.chunk) {
            callbacks.onResponseChunk(data.chunk);
        } else if (data.full_response) {
            callbacks.onComplete(data);
        }
    }
}
```

```javascript
// frontend/js/app.js

await API.speakAndRespondStream(audioBlob, language, difficulty, scenario, sessionId, {
    onTranscription: (data) => {
        // Show "You: [text]"
        this.addMessageElement('user', data.text);
    },
    onResponseChunk: (chunk) => {
        // Append chunk to message bubble
        fullResponse += chunk;
        messageElement.textContent = fullResponse;
    },
    onComplete: (data) => {
        // Play TTS
        this.speakText(data.full_response, language);
    }
});
```

## Benefits Summary

### 1. **Faster Perceived Response Time**
- Users see feedback immediately (0.5-1s vs 3-5s)
- Response appears progressively (feels natural)

### 2. **Better User Experience**
- No more "black box" waiting
- Clear indication of progress
- More engaging interaction

### 3. **Efficient Resource Usage**
- Models stay hot (no reload overhead)
- Streaming uses less memory than buffering
- Better CPU/GPU utilization

### 4. **Scalability**
- Can handle more concurrent users
- Lower latency per request
- Better throughput
